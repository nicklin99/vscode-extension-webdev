# vscode Web项目管理

## 项目简介

Web项目管理,提供类似简化版的idea的项目管理，以增强项目管理体验。

## 功能

该插件通过界面提供以下功能：
- Web项目管理, 打开项目、复制项目路径、打开项目文件夹、关键词过滤
- 创建项目,支持 vue、react、nuxtjs、nextjs、nodejs
